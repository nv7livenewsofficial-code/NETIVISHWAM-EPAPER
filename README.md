# 📰 NETIVISHWAM EPAPER - Digital News Portal

**NETIVISHWAM EPAPER** is a high-performance, responsive static web application designed for digital newspaper publishing. It allows Admins and Reporters to upload, crop, and publish news articles directly from mobile or desktop devices.

### 📱 User Experience
- **Fully Responsive:** Optimized for Mobile, Tablet, and Desktop viewing.
- **HD Content Rendering:** Uses high-quality canvas rendering to ensure text remains sharp.
- **Social Sharing:** One-click sharing options for **WhatsApp**, **Facebook**, and **Copy Link**.
- **Modern UI:** Clean, grid-based layout for easy news discovery.

### 💻 Technical Stack
- **Frontend:** HTML5, CSS3 (Bootstrap 5), JavaScript (ES6).
- **Libraries:**
  - `Cropper.js`: Professional image cropping.
  - `Fabric.js`: Interactive canvas for e-paper mapping.
  - `Lucide Icons`: Modern, lightweight SVG icons.
- **Database:** `LocalStorage` simulation (Works perfectly on GitHub Pages without a backend).
